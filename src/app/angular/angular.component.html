Angular

Instalación
Angular y CLI

Primero ingresamos al sitio de NodeJS para instalarlo, el botón contiene el link.

Link NodeJS

Luego en la terminal utilizaresmos el comando npm install -g @angular/cli para instalar angular cli.

Creación de componentes

Una vez creado nuestro proyecto crearemos componentes con el comando ng generate component "nombre del componente" de este modo se generan una serie de archivos que le corresponden.

Comandos básicos
ng new "nombre-del-proyecto

Con este comando generamos un proyecto nuevo con el framework.

ng serve --open

De este modo podemos ejecutar nuestro script para visualizarlo en un navegador.

npm audt fix

Este es un comando de node que pemirte revisar las vulnerabilidades reportadas en las dependecias del proyecto e intenta solucionarlas.

ng help

Es un comando de ayuda para ver los comandos disponibles.

Angular

Angular es un framework para desarrollo web. Se utiliza para la creación de aplicaciones web de una sola página (SPA) y aplicaciones web dinámicas, usandoo TypeScript como lenguaje principal.

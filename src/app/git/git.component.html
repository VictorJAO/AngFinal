Comandos de Git

  • 1. git init

    Inicializa el repositorio, convirtiendo el directorio en repositorio Git.

  • 2. git add "nombre de archivo" ; git add *

    Con este comando se preparan los archivos que cargaremos a nuestro repositorios.

  • 3. git commit -m "Mensaje"

    Guarda los cambios que se hayan realizado aclarandolo en un mensaje entre "".

  • 4. git remote add origin "URL"

    Conecta el directorio local con un repositorio remoto.
